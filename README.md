# Computational_SSP_Notebook
A notebook in Computational Solid State Physics solving basic to advanced problems.

- Band structure
  - Band structure calculations in reduced dimensions
  - Implemented using the Fourier Grid Hamiltonian (FGH) method

- Crystallography
  - FCC lattice
  - Brillouin zones
